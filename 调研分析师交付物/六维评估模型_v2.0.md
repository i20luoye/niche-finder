# 六维评估模型 v2.0

## 一、模型升级说明

从v1.0的单一"蓝海发现模式"，扩展为**三种发现模式**，每种模式对应不同的评估逻辑和权重配比：

| 模式 | 核心逻辑 | 适用场景 | 代表案例 |
|------|----------|----------|----------|
| 🟢 蓝海模式 | 找无人涉足的全新赛道 | 技术变革催生的新机会 | AI宠物殡葬数字纪念 |
| 🟡 红海细分模式 | 在红海中找未被满足的细分人群/场景 | 大赛道里的垂直小切口 | AI本地商家内容生成（县城市场） |
| 🔵 跨界机会模式 | 两个+赛道交叉产生的新物种 | 成熟领域+AI的跨界融合 | AI银发族短视频创作助手 |

## 二、六维评估标准（通用版）

### 1. 需求热度（基础权重：25%）
**打分标准：**
- 9-10分：月搜索量>10万，增速>50%/季度，用户主动求解决方案
- 7-8分：月搜索量3-10万，增速20-50%/季度，有明确痛点表达
- 5-6分：月搜索量1-3万，增速0-20%/季度，需求存在但不迫切
- 3-4分：月搜索量<1万，增速下滑，伪需求嫌疑大
- 1-2分：几乎没人搜，纯自嗨型需求

**数据来源：** Google Trends、百度指数、微信指数、知乎/小红书话题量

### 2. 竞争烈度（基础权重：20%）
**打分标准：**
- 9-10分：几乎没有直接竞品，大公司完全没布局
- 7-8分：1-3个小玩家，都没形成壁垒，产品体验差
- 5-6分：有5-10个玩家，头部占比<30%，仍有进入空间
- 3-4分：玩家众多，头部效应明显，获客成本高
- 1-2分：巨头垄断，新玩家几乎没有机会

**数据来源：** GitHub stars、应用商店排名、SimilarWeb流量、企查查

### 3. 变现能力（基础权重：20%）
**打分标准：**
- 9-10分：付费意愿极强，客单价>500元/年，可验证的付费场景
- 7-8分：付费意愿较强，客单价100-500元/年，有成熟付费模式
- 5-6分：有一定付费意愿，客单价50-100元/年，需要教育市场
- 3-4分：付费意愿弱，客单价<50元/年，只能靠广告/流量变现
- 1-2分：用户完全不愿意付费，纯公益性质

**数据来源：** 竞品定价、用户调研、电商销量、行业报告

### 4. 进入门槛（基础权重：15%）
**打分标准：**
- 9-10分：零门槛，有手就能做，纯信息差/创意
- 7-8分：低门槛，会用AI工具+基础开发就能做
- 5-6分：中等门槛，需要特定行业知识或技术能力
- 3-4分：较高门槛，需要团队、资金、资源积累
- 1-2分：极高门槛，需要牌照、资质、巨头资源

**数据来源：** 技术栈复杂度、行业资质要求、启动资金需求

### 5. 风险等级（基础权重：10%）
**打分标准：**
- 9-10分：几乎零风险，不违法不违规，大平台不封禁
- 7-8分：低风险，轻微合规风险，可通过设计规避
- 5-6分：中等风险，有政策/平台/法律不确定性
- 3-4分：较高风险，踩红线概率大，可能被封号/罚款
- 1-2分：极高风险，直接违法，随时可能进去

**数据来源：** 法律法规、平台规则、行业政策

### 6. 增长潜力（基础权重：10%）
**打分标准：**
- 9-10分：指数级增长空间，天花板>10亿/年市场规模
- 7-8分：快速增长空间，天花板1-10亿/年
- 5-6分：线性增长空间，天花板1000万-1亿/年
- 3-4分：增长空间有限，天花板100-1000万/年
- 1-2分：几乎没有增长空间，小生意，赚点零花钱

**数据来源：** 行业报告、市场规模测算、用户生命周期价值

**设计逻辑：**
- **蓝海模式**：最看重竞争烈度（找没人做的），其次是需求真实性
- **红海细分模式**：最看重需求热度（细分人群够不够大）和变现能力（能不能赚钱）
- **跨界机会模式**：风险占比更高（跨界容易踩坑），需求验证权重降低（全新品类）

## 四、三种发现模式的操作方法论

### 🟢 蓝海模式：从技术演进找机会
**操作步骤：**
1. 扫描最新AI技术突破点（多模态、Agent、RAG等）
2. 列出该技术能解决的10个真实痛点
3. 过滤掉已经有成熟产品的方向
4. 剩下的按"需求真实度×技术成熟度"排序

**信号源：** GitHub Trending、Hacker News、Product Hunt、arXiv

### 🟡 红海细分模式：从人群/场景切分找机会
**操作步骤：**
1. 选择一个大赛道（如AI写作、AI绘画）
2. 按人群维度拆分（年龄/职业/地域/收入）
3. 按场景维度拆分（工作/生活/学习/娱乐）
4. 交叉筛选出"人少钱多"的细分格子

**信号源：** 电商评论区、小红书/知乎话题、社群讨论、竞品差评

### 🔵 跨界机会模式：从赛道交叉找新物种
**操作步骤：**
1. 列出3-5个你熟悉的成熟赛道
2. 每条赛道挑出3个核心痛点
3. 用AI重新思考每个痛点的解决方案
4. 交叉组合出"赛道A × 赛道B × AI"的新物种

**信号源：** 跨界案例、行业报告、不同领域KOL观点
