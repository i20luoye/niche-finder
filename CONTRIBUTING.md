# Contributing to Niche Finder

First off, thanks for taking the time to contribute! 🎉

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please create an issue and include:
- A clear and descriptive title
- Steps to reproduce the bug
- Expected behavior vs actual behavior

### Suggesting Enhancements
Have an idea for a new feature? We'd love to hear it!
- Use the issue tracker to propose new features.
- Provide a clear use case for the feature.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Issue that pull request!

## Code Style
Please follow the existing code style. For Python code, we loosely follow PEP 8.
