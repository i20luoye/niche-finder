## 四、核心模块详解

### 4.1 双引擎策略（核心设计）

系统采用**主引擎 + 后备引擎 + 兜底**的三层架构：

| 层级 | 引擎 | 耗时 | 适用场景 |
|------|------|------|---------|
| **主引擎** | LightweightEngine | 75-100秒 | MVP日常使用，快速出报告 |
| **后备** | CrewAI多Agent | 5-8分钟 | 深度分析模式，付费增值 |
| **兜底** | Mock数据 | 即时 | API不可用时的降级方案 |

**降级逻辑**（在 `server.py` 的 `run_analysis_task` 函数中）：
```python
try:
    # 1. 优先使用轻量引擎
    engine = LightweightEngine()
    result = engine.run(mode, topic, user_profile)
except Exception:
    try:
        # 2. 降级到CrewAI
        result = run_crew_analysis(mode, topic, user_profile)
    except Exception:
        # 3. 兜底Mock
        result = generate_mock_report(mode, topic)
```

### 4.2 LightweightEngine（主引擎）

**文件**: `niche_finder/engine/lightweight.py` (376行)

**核心流程** — 两阶段分析：

```
阶段1: 市场数据搜索 (Tavily)
  ├── 蓝海模式: 3个搜索查询 (空白市场/新兴需求/低竞争)
  ├── 红海细分: 3个搜索查询 (市场痛点/细分需求/未满足用户)
  ├── 跨界模式: 3个搜索查询 (跨界融合/技术应用/新兴场景)
  └── 深度分析: 4个搜索查询 (更全面的市场数据)

阶段2: LLM分析生成 (日日新API)
  ├── 机会发现: 1次LLM调用 → 生成3-5个机会点
  └── 评审排序: 1次LLM调用 → 六维评分 + 排序 + 行动方案
```

**关键方法**:
- `run(mode, topic, user_profile)`: 主入口，返回完整报告
- `_search_market_data(mode, topic)`: Tavily搜索
- `_discover_opportunities(topic, search_results, mode, user_profile)`: LLM机会发现
- `_evaluate_and_rank(opportunities, topic, search_results)`: LLM评审排序
- `_generate_report(...)`: 格式化输出

**4种模式的搜索策略**:
- `blue_ocean`: 搜索"空白市场/新兴需求/低竞争领域"
- `red_ocean_niche`: 搜索"市场痛点/细分需求/未满足用户群"
- `cross_domain`: 搜索"跨界融合机会/技术应用场景/新兴场景"
- `deep_analysis`: 更全面的4个搜索查询

### 4.3 CrewAI引擎（后备）

**文件**: `niche_finder/crew.py` (177行), `niche_finder/agents/niche_agents.py` (148行)

**6个专业Agent**:
1. **机会发现专家** (max_iter=2): 识别潜在利基市场机会
2. **数据分析师** (max_iter=2): 收集和分析市场数据
3. **竞品分析师** (max_iter=2): 评估竞争格局
4. **市场研究员** (max_iter=2): 深入市场调研
5. **产品策略师** (max_iter=2): 评估商业可行性
6. **首席评审官** (max_iter=2): 综合评审，输出最终报告

**Crew编排** (`niche_finder/crew.py`):
- `NicheCrew` 类管理Agent组合和任务流程
- 按模式选择不同的Agent组合和任务序列

**已知问题**: CrewAI在300秒内无法完成端到端测试，max_iter已从5/4/3降到2，仍超时。当前作为后备引擎。

### 4.4 发现模式 (`niche_finder/discovery_modes.py`)

3种核心模式 + 1种深度模式：
- `blue_ocean_mode`: 蓝海发现 — 寻找竞争少的新兴市场
- `red_ocean_niche_mode`: 红海细分 — 在红海中找细分机会
- `cross_domain_mode`: 跨界创新 — 跨行业/跨领域的创新机会
- `deep_analysis_mode`: 深度分析 — 全面深度研究（CrewAI模式）
